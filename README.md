# RAG-DOC

**RAG-DOC** — это RAG-ассистент (Retrieval-Augmented Generation) для ваших документов. Он обеспечивает семантический поиск и позволяет задавать вопросы по содержимому локальных файлов, используя мощь LLM и векторных эмбеддингов.

## 🚀 Возможности

- 🔍 **Семантический поиск**: Умный поиск по вашим файлам на основе смысла, а не только по ключевым словам.
- 💬 **Вопросно-ответная система**: Задавайте вопросы естественным языком и получайте точные ответы, сгенерированные LLM на основе ваших же документов.
- 📄 **Поддержка различных форматов**: Парсинг PDF и других документов (используются `docling` и `pypdf`).
- ⚡ **Высокая производительность**: Работает на связке LangChain, Qdrant (для векторного хранения) и OpenAI.
- 🛠 **Современный Python стек**: Строгая типизация (`mypy`), линтинг (`ruff`, `black`) и быстрая сборка зависимостей с помощью `uv`.

## 📦 Стек технологий

- **Язык**: Python 3.11
- **Пакетный менеджер**: [uv](https://github.com/astral-sh/uv)
- **LLM & Оркестрация**: LangChain, OpenAI API
- **Векторная база данных**: Qdrant
- **ML & Тензоры**: PyTorch (CUDA 12.6)

## ⚙️ Установка и запуск

### 1. Предварительные требования
Для работы проекта вам потребуется:
- Установленный **Python 3.11**
- Пакетный менеджер [uv](https://docs.astral.sh/uv/)

### 2. Клонирование репозитория
```bash
git clone https://github.com/Sh1nroq/RAG-DOC.git
cd RAG-DOC
```

### 3. Установка зависимостей
Проект настроен на использование `uv` для быстрого разрешения зависимостей.

```bash
# Инициализация виртуального окружения и установка пакетов
uv sync
```
*Или, если вы устанавливаете глобально/в существующее окружение:*
```bash
uv pip install -e .
```

### 4. Настройка переменных окружения
Для работы с языковыми моделями (OpenAI) необходимо задать API ключ. 

1. Скопируйте пример файла конфигурации:
   ```bash
   cp .env.example .env
   ```
2. Откройте файл `.env` и вставьте ваш ключ от OpenRouter:
   ```dotenv
   OPENAI_API_KEY=sk-your_api_key_here
   ```

## 🛠 Разработка

Проект использует современные инструменты для поддержания чистоты кода:

```bash
# Форматирование кода
ruff formant ./src/ --fix
ruff check --fix src/

# Проверка типов
mypy ./src/
```
